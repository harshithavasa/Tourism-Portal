# Tourism-Portal

Explore World

Explore World is a static tourism website built using only HTML and CSS. It offers a simple and visually appealing interface that showcases destinations, tours, and user testimonials while allowing visitors to explore travel inspiration through an intuitive design.

Features: 
Home section introducing the website and travel theme

Navigation bar for easy access to all sections

Destinations section highlighting popular travel spots with images and short details

Tours section showcasing travel packages or featured trips

Testimonials section displaying traveler feedback

Booking form for collecting travel details (static, non-functional)

Footer with social media icons and embedded map icon

Technologies Used:
HTML5 for content structure

CSS3 for layout, styling, and responsiveness
